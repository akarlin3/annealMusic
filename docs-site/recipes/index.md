# The AnnealMusic Recipe Book

The Recipe Book contains **16 fully working, self-contained scripts and guides** spanning generative art, hardware integration, music cognition, machine learning, and computational physics.

Each recipe contains a complete walkthrough, prose explanation, expected outputs, and direct links to execute or verify them.
