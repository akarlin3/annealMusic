# AnnealMusic API & Schema Stability Commitments: v7 Closeout

To guarantee that research projects, clinical trials, and data sonifications initiated during the v7 cycle remain fully operational and reproducible for years, AnnealMusic establishes formal **API and Schema Stability Commitments**.

All components documented below are frozen and backwards-compatible. Any future updates (in v8 or beyond) must respect these interfaces or provide documented, automatic translation scripts.

---

## 1. Relational Database Schemas

The following database tables are **locked and stable**:

- **Study Collaboration Model**: `studies`, `study_investigators`, `study_resources`, `study_versions`, and `study_audit_log` tables. New columns can only be added additively with nullable defaults.
- **Accessibility Systems**: The `accessibility_descriptions` table layout is fully stable.
- **Participant Logging & Progress**: `lesson_progress` table and the associated `step_actions` schema support backwards-compatible, additive progress tracking.

---

## 2. Python Sandbox & Scripting Bindings

The isolated CPython worker sandbox (`pyodide-worker.js`) running Python `3.12` maintains a locked, whitelisted runtime space:

- **The `anneal` Library**: Methods for parameter manipulation (`state.get()`, `state.set()`), spectral observations (`engine.get_spectrum()`), datalogger controls (`datalog.start()`, `datalog.stop()`), and citation retrieval (`anneal.cite()`) are frozen.
- **Matplotlib WebAgg Backend**: The virtual plotting interface translating `plt.show()` into PNG stream packages is stable.
- **CDN Preloading Allowlist**: Package versions for `numpy`, `scipy`, `matplotlib`, `pandas`, and `scikit-learn` are locked in GHA verification suites.

---

## 3. Session Datalogger & Serialization Formats

The in-browser high-performance datalogger ring buffer and serializers are frozen:

- **JSON Lines (JSONL)**: High-resolution temporal tick outputs use the locked key-value schema.
- **Tabular CSV Flattener**: The columns generated by the dot-notation flattener (`tools/cli/src/output/converter.ts`) are static and guaranteed to align with down-stream PyArrow and Pandas parquet converters.

---

## 4. Open Sound Control (OSC) Address Space

The bidirectional OSC address namespace is frozen:

- **Observability Addresses**: Broadcast channels `/anneal/state/*` and `/anneal/engine/*` emit consistent schemas.
- **Control Targets**: Incoming control inputs via `/control/slider/*` and `/control/trigger/*` are fully supported.
- **WebSocket-OSC Bridge**: The JSON-RPC bridge schema is stable.
