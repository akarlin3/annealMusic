## 2. Edge Case Inventory & Resolution Approach

- **Cross-mode notifications**:
  - _Problem_: A Meditation user receives a technical render-completed toast.
  - _Resolution_: Update the global `showToast` in `src/pages/App.tsx` to inspect `appMode`. Suppress technical terms (`"Export"`, `"sonification"`, `"MIDI"`, `"Collab"`, `"AI Modified"`) when in Meditation mode.
- **Mid-task mode switches**:
  - _Problem_: Musician sculpt state is lost on path changes or mode redirects.
  - _Resolution_: Automatically persist parameter store changes to `localStorage` under `am_unsaved_patch_state`. Auto-hydrate parameter values on page load if hash is absent. Back up and restore user's sub-mode (`sketch` or `drone`).
- **Mode-switch-during-playback**:
  - _Verification_: Confirm multi-tab connection via `BroadcastChannel` bridge functions cleanly without issues. Document this constraint for single-tab transitions.
- **Embedded views and mode**:
  - _Verification_: Confirm `/embed/<slug>` and `/embed-figure/<slug>` entry points don't load the `ModeProvider` or `ModeAesthetic` context, ignoring modes entirely.
- **Deep-linked navigation**:
  - _Verification_: Verify `/learn` deep links bypass active track mode filters using the raw tracks catalog.
- **Lesson recommendations crossing modes**:
  - _Resolution_: Update `NextLessonPicker.tsx` to match recommended lesson modes against the active mode, appending `"This lesson is also relevant to Musician mode"` for cross-mode recommendations.

## 3. Accessibility Regression Test Plan

- **Contrast Ratios**: Check foreground/background contrast ratios against WCAG 2.1 AA (>= 4.5:1) for Meditation, Musician, and Researcher themes.
- **Keyboard Navigation**: Focus indicators (`focus-visible:ring-1 focus-visible:ring-amber-500`) active on all newly refactored input/select inputs.
- **Reduced Motion**: Verify CSS animations and transitions disable globally under `prefers-reduced-motion: reduce`.

## 4. Performance Verification Plan

- **Switch Latency**: Verify target mode switch latency remains `<200ms`.
- **Bundle Analysis**: Keep new primitives slim to preserve cold start metrics.
