# Anneal Ambiance v1.9 · WebGL Visualization Upgrade · Implementation Plan

This plan details the upgrade of the v0.1 canvas visualizer to a modern, shader-based WebGL2 visualizer with full canvas fallback.

---

## 1. Unified `VisualRenderer` Interface

We define a clear contract to make the React visualizer shell completely renderer-agnostic:

```typescript
export interface LoopRing {
  slot: number; // 0..2
  level: number; // 0..1
  frozen: boolean;
}

export interface VisualState {
  w: number; // CSS width of visualizer area
  h: number; // CSS height of visualizer area
  dt: number; // Clamp delta time in seconds
  phases: number[]; // Orbit phases of partials (advanced in-place)
  freqs: number[]; // Frequency (Hz) per partial
  count: number; // Particle count
  spectrum: Uint8Array | null; // FFT spectrum
  sampleRate: number; // AudioContext sample rate
  fftSize: number; // Analyser FFT size
  inputLevel?: number; // Mic level (0..1)
  loops?: LoopRing[]; // Loop slots
}

export interface VisualRenderer {
  mount(canvas: HTMLCanvasElement): void;
  unmount(): void;
  resize(width: number, height: number, dpr: number): void;
  drawFrame(state: VisualState, now: number): void;
  setQuality(level: 'low' | 'medium' | 'high'): void;
  dispose(): void;
}
```

## 2. Library Selection: `ogl`

We will use `ogl` for our WebGL abstraction layer.

- **Footprint:** ~6KB gzipped, leaving the bundle size extremely small.
- **Abstractions:** Thin classes wrapping WebGL2 objects (`Renderer`, `Geometry`, `Program`, `Mesh`, `Texture`).
- **Control:** Allows easy binding of low-level variables, precision tuning, and robust recovery from `webglcontextlost` events.

---

## 3. Shader Architecture

We utilize a **Single-Pass Full-Screen Quad Fragment Shader**. This architecture compiles a single, fast fragment shader that evaluates all visual artifacts procedurally for each pixel.

### Data Channels

- **Partials Data:** Passed as a uniform array `u_partials[16]` of `vec4` (x, y, freq, amplitude).
- **Spectrum Data:** Transferred as a dynamic 512x1 texture containing red-channel (`R8`) bytes.

### Shader Uniforms

```glsl
uniform vec2 u_resolution;       // Viewport size in pixels (w, h)
uniform float u_dpr;             // Device pixel ratio
uniform float u_time;            // Elapsed time
uniform float u_base_radius;     // Base orbit radius (px)
uniform float u_rms;             // Global RMS amplitude (0..1)
uniform float u_input_level;     // Input voice amplitude (-1.0 if disconnected)
uniform int u_partial_count;     // Particle count
uniform vec4 u_partials[16];     // Packed orbits (x, y, freq, amp)
uniform sampler2D u_spectrum;    // 1D spectrum texture (512x1)
uniform int u_show_spectrum;     // Toggle trace render
uniform vec3 u_loop_levels;      // Loop slot levels (0..1)
uniform vec3 u_loop_frozen;      // Loop slot frozen status (0.0 or 1.0)
```

## 4. GLSL Code Drafts

### Vertex Shader (`visual.vert.glsl`)

```glsl
#version 300 es
in vec2 position;
in vec2 uv;
out vec2 v_uv;

void main() {
    v_uv = uv;
    gl_Position = vec4(position, 0.0, 1.0);
}
```

### Fragment Shader (`visual.frag.glsl`)

```glsl
#version 300 es
precision highp float;

in vec2 v_uv;
out vec4 fragColor;

uniform vec2 u_resolution;
uniform float u_dpr;
uniform float u_base_radius;
uniform float u_rms;
uniform float u_input_level;
uniform int u_partial_count;
uniform vec4 u_partials[16];
uniform sampler2D u_spectrum;
uniform int u_show_spectrum;
uniform vec3 u_loop_levels;
uniform vec3 u_loop_frozen;

const vec3 COLOR_AMBER = vec3(245.0 / 255.0, 158.0 / 255.0, 11.0 / 255.0);
const vec3 COLOR_ROSE  = vec3(248.0 / 255.0, 113.0 / 255.0, 113.0 / 255.0);
const vec3 COLOR_CYAN  = vec3(125.0 / 255.0, 211.0 / 255.0, 252.0 / 255.0);
const vec3 COLOR_CORE  = vec3(254.0 / 255.0, 215.0 / 255.0, 170.0 / 255.0);
const vec3 COLOR_MID   = vec3(251.0 / 255.0, 191.0 / 255.0, 36.0 / 255.0);
const vec3 COLOR_EDGE  = vec3(251.0 / 255.0, 146.0 / 255.0, 60.0 / 255.0);

const float PI = 3.14159265359;

float distanceToSegment(vec2 p, vec2 a, vec2 b) {
    vec2 pa = p - a, ba = b - a;
    float h = clamp(dot(pa, ba) / dot(ba, ba), 0.0, 1.0);
    return length(pa - ba * h);
}

void main() {
    vec2 fragCoord = v_uv * u_resolution;
    vec2 center = u_resolution * 0.5;
    float dist = distance(fragCoord, center);

    // 1. Background color
    vec3 color = vec3(12.0 / 255.0, 10.0 / 255.0, 9.0 / 255.0);

    // 2. Central Halo
    float haloRadius = u_base_radius * 1.6 * (1.0 + u_rms * 0.15);
    float haloT = smoothstep(haloRadius, 0.0, dist);
    color += COLOR_AMBER * (0.04 + u_rms * 0.04) * haloT;

    // 3. Faint Input Ring
    if (u_input_level >= 0.0) {
        float ringR = u_base_radius * 1.6 * (1.0 + u_input_level * 0.25);
        float ringAlpha = 0.05 + u_input_level * 0.18;
        float ringDist = abs(dist - ringR);
        float ringEdge = smoothstep(1.25 * u_dpr, 0.25 * u_dpr, ringDist);
        color += COLOR_MID * ringAlpha * ringEdge;
    }

    // 4. Loop Rings
    float loopRadii[3] = float[](1.78, 1.98, 2.18);
    vec3 loopColors[3] = vec3[](COLOR_AMBER, COLOR_ROSE, COLOR_CYAN);
    float angle = atan(fragCoord.y - center.y, fragCoord.x - center.x);
    if (angle < 0.0) angle += 2.0 * PI;

    for (int slot = 0; slot < 3; slot++) {
        float lvl = u_loop_levels[slot];
        if (lvl <= 0.0 && u_loop_frozen[slot] == 0.0) continue;

        float radius = u_base_radius * loopRadii[slot] * (1.0 + lvl * 0.18);
        float ringAlpha = 0.06 + lvl * 0.22 + (u_loop_frozen[slot] > 0.0 ? 0.06 : 0.0);
        float ringDist = abs(dist - radius);
        float ringEdge = smoothstep(1.25 * u_dpr, 0.25 * u_dpr, ringDist);

        float start = (float(slot) * 2.0 * PI) / 3.0;
        float sweep = (u_loop_frozen[slot] > 0.0) ? 2.0 * PI : 1.7;
        float relativeAngle = mod(angle - start, 2.0 * PI);

        if (relativeAngle <= sweep || (2.0 * PI - relativeAngle) <= 0.01) {
            color += loopColors[slot] * ringAlpha * ringEdge;
        }
    }

    // 5. Ambient orbits
    for (int i = 0; i < u_partial_count; i++) {
        vec4 p = u_partials[i];
        float amp = p.w;
        float pDist = distance(fragCoord, p.xy);
        float glowRadius = (5.0 + amp * 22.0) * u_dpr;

        if (pDist < glowRadius) {
            float t = pDist / glowRadius;
            vec3 pColor;
            float pAlpha;

            if (t < 0.4) {
                float n = t / 0.4;
                pColor = mix(COLOR_CORE, COLOR_MID, n);
                pAlpha = mix(0.55 + amp * 0.35, 0.3 + amp * 0.25, n);
            } else {
                float n = (t - 0.4) / 0.6;
                pColor = mix(COLOR_MID, COLOR_EDGE, n);
                pAlpha = mix(0.3 + amp * 0.25, 0.0, n);
            }
            color += pColor * pAlpha;
        }
    }

    // 6. Spectrum trace
    if (u_show_spectrum == 1) {
        float bottomPad = 10.0 * u_dpr;
        float spectrumHeight = 36.0 * u_dpr;
        float bottomY = u_resolution.y - bottomPad;
        float usableFrac = 0.45;
        float normX = fragCoord.x / u_resolution.x;

        if (normX <= 1.0) {
            float texCoord = normX * usableFrac;
            float val = texture(u_spectrum, vec2(texCoord, 0.5)).r;
            float traceY = bottomY - val * spectrumHeight;

            float stepW = 1.0 / u_resolution.x;
            float valPrev = texture(u_spectrum, vec2(max(0.0, normX - stepW) * usableFrac, 0.5)).r;
            float valNext = texture(u_spectrum, vec2(min(1.0, normX + stepW) * usableFrac, 0.5)).r;

            vec2 pPrev = vec2(fragCoord.x - 1.0, bottomY - valPrev * spectrumHeight);
            vec2 pCurr = vec2(fragCoord.x, traceY);
            vec2 pNext = vec2(fragCoord.x + 1.0, bottomY - valNext * spectrumHeight);

            float d1 = distanceToSegment(fragCoord, pPrev, pCurr);
            float d2 = distanceToSegment(fragCoord, pCurr, pNext);
            float minD = min(d1, d2);

            float traceAlpha = smoothstep(1.0 * u_dpr, 0.0 * u_dpr, minD);
            color += vec3(245.0 / 255.0, 245.0 / 255.0, 244.0 / 255.0) * 0.16 * traceAlpha;
        }
    }

    fragColor = vec4(color, 1.0);
}
```

## 5. Capability Probe & Fallback Tree

Capability detection runs once on first load:

```typescript
export interface CapabilityResult {
  webgl_supported: boolean;
  reason?: string;
}

export function probeWebGL2(): CapabilityResult {
  if (typeof window === 'undefined') {
    return { webgl_supported: false, reason: 'SSR' };
  }
  try {
    const canvas = document.createElement('canvas');
    const gl = canvas.getContext('webgl2', {
      alpha: false,
      depth: false,
      stencil: false,
      antialias: true,
    });
    return { webgl_supported: !!gl };
  } catch (e) {
    return {
      webgl_supported: false,
      reason: e instanceof Error ? e.message : 'Unknown',
    };
  }
}
```

## 6. Resolution Scaling & Preset Configs

### Dynamic Throttling:

- **DPR Limit:** Up to 2x (ignores higher DPRs like 3x to protect fill-rate).
- **Quality Ceiling:**
  - **High:** Scale viewport at 1.0x ceiling.
  - **Medium:** Scale viewport at 0.8x ceiling.
  - **Low:** Scale viewport at 0.6x ceiling.
- **Dynamic Framerate Monitor:**
  - Measure frame duration.
  - Maintain a rolling window of 60 frames.
  - If average > 18ms (less than 60 FPS), scale resolution down by `0.85x` (min `0.5x`).
  - If average < 12ms and downscaled, scale back up by `1.1x` (up to quality ceiling).

---

## 7. visibility and Audio Throttling

- **Page Hidden:** If `document.visibilityState === 'hidden'`, rendering pauses entirely.
- **Silence (RMS < 0.001 and isPlaying is false):** Throttle visualizer to **5 FPS** to save battery.

---

## 8. Mobile Integration & WebGL Context Loss

- **Power Preference:** Use `powerPreference: 'low-power'` to prioritize battery life.
- **Context Loss Handling:** Bind to `webglcontextlost` and `webglcontextrestored`. Recover meshes and textures immediately, or fallback to Canvas if loss happens repeatedly.
