# Goal Description

Let users upload their own audio files to use as granular engine sources. Uploaded sources live in object storage, are referenced by ID like captures, and can be saved in patches and shared via gallery. The curated bank from v0.9 continues to ship as the default set; user sources extend it.

This is the first slice that puts arbitrary user audio on the platform's storage in a way that's shareable through the gallery.

---

## User Review Required

> [!IMPORTANT]
>
> - **designated bundled source fallback:** If a user source is flagged by a moderator or deleted by the owner, any public patch referencing it will fall back to **`b:glasspad`** (the default ambient pitched source at index 0), displaying a non-blocking toast warning to the loader.
> - **dedicated render-only endpoint:** For preview rendering compatibility, the headless browser must be able to load shared user sources anonymously. We implement `GET /api/v1/render/user-sources/:id` which strictly allows reads of `shared` sources without authentication, avoiding the leakage of admin credentials.
> - **amplitude/DC/silence/noise threshold checks:** The server enforces rigorous checks during decoding to reject silent uploads (peak amplitude < -40dB), invalid DC offsets (offset > 10% of full-scale), and high-frequency hiss/noise (zero-crossing rate > 45%).

### Backend Endpoints & Audio Pipeline

#### [NEW] [user_sources.py](file:///Users/averykarlin/annealMusic/api/app/routers/user_sources.py)

- Implement endpoints:
  - `POST /api/v1/user-sources` — Multipart upload. Implements quota checks, display name moderation screening, ffmpeg-based decoding & transcode.
  - `GET /api/v1/user-sources/me` — Lists user's owned sources.
  - `GET /api/v1/user-sources/:id` — Follows visibility rules: permits if owner or if `shared`; returns 451 if `flagged`; 404 otherwise. Redirects 302 to storage presigned URL.
  - `PATCH /api/v1/user-sources/:id` — Updates `display_name` with filename screening.
  - `DELETE /api/v1/user-sources/:id` — Hard deletes from S3 + DB. Decrements `source_count` & `bytes_used`.
- Dedicated render-only endpoint:
  - `GET /api/v1/render/user-sources/:id` — Gated exclusively to `visibility == 'shared'`.

- Add transcode functions:
  - `decode_any_to_wav(data: bytes) -> bytes` (decodes WAV/MP3/FLAC/AAC/OGG/Opus to signed 16-bit PCM WAV).
  - `transcode_to_mono_opus(wav_bytes: bytes, bitrate_kbps: int) -> bytes` (mono transcoding).
- Add analytical validation check functions:
  - `validate_audio_safety(wav_bytes: bytes) -> None` (parses PCM samples to enforce peak threshold, DC offset check, and zero-crossing noise density check).
