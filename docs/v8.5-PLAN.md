## 3. Risks & Mitigations

| Risk Identified                  | Impact | Likelihood | Mitigation Strategy                                                                                                                                        |
| :------------------------------- | :----- | :--------- | :--------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Playwright Web Audio Mocking** | High   | Medium     | Playwright tests will mock the Web Audio Context and audio element events to avoid external audio interface locks or timing variations in headless setups. |
| **Zenodo Sandbox Availability**  | Medium | Low        | ZENODO API calls will be fully mocked in integration tests, and sandbox tests will fall back gracefully to a mock record if no key exists.                 |
| **Pyodide Sandbox Load Time**    | Medium | Medium     | Pre-load Pyodide inside the e2e browser context or mock the Pyodide runtime script load using cached scripts.                                              |
