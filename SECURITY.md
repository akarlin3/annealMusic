# Security Policy (SECURITY.md)

## Supported Versions

Only the latest release of AnnealMusic is actively supported with security updates:

| Version  | Supported              |
| -------- | ---------------------- |
| v8.3.0   | :white_check_mark: Yes |
| < v8.3.0 | :x: No                 |

---

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please **do not open a public issue**. Instead, report it privately:

- **Email Contact:** security@annealmusic.com
- We will acknowledge receipt of your report within 48 hours.
- We aim to coordinate a release resolving verified reports within 30 days of disclosure.

---

## Bug Bounty Program (Stub)

> AnnealMusic does not currently offer paid bug bounties. We acknowledge responsible disclosure publicly (with permission) in our [Security Acknowledgments documentation](docs/SECURITY_ACKNOWLEDGMENTS.md). We may introduce a paid bounty in the future as the project matures.
